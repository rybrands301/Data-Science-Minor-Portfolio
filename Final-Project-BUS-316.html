flight_time_intervals <- flight_data %>%
  mutate(time = case_when(
    sched_dep_time >= 0001 & sched_dep_time <= 0600 ~ "12:01am-6am",
    sched_dep_time >= 0601 & sched_dep_time <= 1200 ~ "6:01am-12pm",
    sched_dep_time >= 1201 & sched_dep_time <= 1800 ~ "12:01pm-6pm",
    sched_dep_time >= 1801 & sched_dep_time <= 2400 ~ "6:01pm-12am"
    ))

Explanation 1) Here I created the new dataframe flight_time_intervals 2) I used mutate to create a new variable, time, to represent the different time intervals 3) I split up the intervals with case_when so that there were 4 time intervals covering all 24 hours of the day (as requested)

flight_time_intervals %>%
  drop_na(dep_delay) %>%
  group_by(time) %>%
  mutate(departure_delayed = ifelse(dep_delay>0,    "delayed","ontime")) %>%
  count(time, departure_delayed) %>%
  mutate(proportion_delayed = n/sum(n)) %>%
  filter(departure_delayed == "delayed") %>%
  arrange(proportion_delayed) %>%

It makes sense that the lowest number of dpartures would be between the 12:01am - 6 am interval since less people would want to catch a flight at that time.
